🌡️ Temperature Control Using Fuzzy Logic

Repository: basheeraltawil/Temperature-control-using-fuzzy-logic

🔍 Project Overview

This project demonstrates how to control temperature using Fuzzy Logic in MATLAB. Instead of traditional on/off control, fuzzy logic mimics human decision-making — offering smoother and more efficient responses.

🧠 Why Fuzzy Logic?

Fuzzy logic is ideal for systems that require flexible, rule-based control — just like how humans adjust temperature based on "cold", "warm", or "hot", not just numbers. This makes it perfect for real-world applications like air conditioners, incubators, and smart thermostats.

🔧 Tools & Technologies

  • MATLAB: For designing and simulating fuzzy logic controllers.
  • .fis File: Contains the fuzzy rules and membership functions.
  • Rule-Based System: Designed for adaptive temperature adjustment.

📁 Key Files

  • temperature_controlling1.fis — The fuzzy inference system file.
  • README.md — Documentation for setup and usage.

💡 Real-World Use Cases

  • Smart HVAC systems
  • Greenhouse climate control
  • Medical devices (e.g., infant incubators)
